#### なぜ本名をここに書かないのか

本名を安易に公開するのは、非常に危険な行為だと考えているからです。漏れたのが住所であれば引っ越せばよくて、それがハンドルネームであれば改名すればよいのですが、本名の場合は簡単に改名するわけにはいかないからです。日本において、犯罪をやらかした場合には実名報道される原則があります。犯罪をやらかすリスクというのは、いくら気を付けていてもゼロではないため、もしそうなった場合に備えて、隠せる場合に本名は隠すことにしています。Facebookだけは、実名登録が原則なので、本名を実名で登録しています。今後、仕事等で本名を公開せざるを得ない場合には、公開するものとします。
